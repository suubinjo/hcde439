OVERVIEW

I work for a holiday light company, and the cheif innovation officer asked me to design a new blinking pattern using at least three LEDs.

SCHEMATIC

For this assignment, I used three red LEDs (2V) and its own resistor so that each LED can be controlled independently. I used a 220立 resistor for each LED.

Here's my calculation:
V_R = V_pin - V_LED = 5V - 2V = 3V
I = V_R / R = 3V / 0.02A = 150立

And then I rounded it up to 220立 resistor which was provided in the kit.

CIRCUIT

ARDUINO

Below is my Arduino code snippet. I used the built in example "Blink" provided by Arduino and edited it to control three LEDs independently. (Source: https://docs.arduino.cc/built-in-examples/basics/Blink/) 


        // the setup function runs once when you press reset or power the board
void setup() {
  // initialize three LEDs as an output.
  pinMode(11, OUTPUT); // first LED
  pinMode(8, OUTPUT); // second LED
  pinMode(5, OUTPUT); // third LED
}

// the loop function runs over and over again forever
void loop() {
  digitalWrite(11, HIGH);  // turn the first LED on (HIGH is the voltage level)
  delay(1000);                      // wait for a second
  digitalWrite(11, LOW);   // turn the first LED off by making the voltage LOW
  delay(1000);
  digitalWrite(8, HIGH);  // turn the second LED on (HIGH is the voltage level)
  delay(1000);                      // wait for a second
  digitalWrite(8, LOW);   // turn the second LED off by making the voltage LOW
  delay(1000);
  digitalWrite(5, HIGH);  // turn the third LED on (HIGH is the voltage level)
  delay(1000);                      // wait for a second
  digitalWrite(5, LOW);   // turn the third LED off by making the voltage LOW
  delay(1000);                    // wait for a second
}
