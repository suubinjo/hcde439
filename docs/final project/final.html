CONCEPT

This assignment asks you to create an interactive webpage that communicates with an Arduino using WebSerial. The webpage must be built with p5.js including both setup() and draw() function. The project requires interaction in both ways (webpage ↔ Arduino), with at least two Arduino input devices and at least one Arduino output device.

SCHEMATIC

For this assignment, I used a 4x4 keypad and a push button as the input devices, and one red LED (with 220Ω resistor) as the output device connected to the Arduino. All grounds in the circuit are connected to the Arduino GND.

Here's my calculation:
V_R = V_pin - V_LED = 5V - 2V = 3V
I = V_R / R = 3V / 0.02A = 150Ω

And then I rounded it up to 220Ω resistor which was provided in the kit. For button, I used a 10kΩ resistor to limit thr current because the value allows only a very small current to flow while still providing a stable signal to the Arduino input pin.
Note: This explanation is adapted from my Assignment 2 webpage.

CIRCUIT

I made a prototype that connects an Arduino keypad, button, and LED to a webpage.

  • Web → Arduino: Pressing the same number key on the laptop keyboard toggles the Arduino LED.
  • Arduino → Web (Keypad): Whe a keypad button is pressed, the webpage displays the keypad input.
  • Arduino → Web (Button): Pressing the button changes the webpage background to a random color.
  • Keyboard reset (Web only): Pressing ENTER resets the background color to white.

ARDUINO

The keypad code part was adapted from my Assignment 5 webpage.


#include <Keypad.h>
// define the number of rows and cols in the keypad
const byte ROWS = 4;
const byte COLS = 4;

// define the key layout corresponding to the keypad buttons
char keys[ROWS][COLS] = {
  {'1','2','3','A'},
  {'4','5','6','B'},
  {'7','8','9','C'},
  {'*','0','#','D'}
};

// define which arduino pins are connected to the keypad row pins and column pins
byte rowPins[ROWS] = {9,8,7,6}; // Arduino pins connected to Row1-Row4
byte colPins[COLS] = {5,4,3,2}; // Arduino pins connected to Col1-Col4

// Initialize an instance of the Keypad class
Keypad keypad = Keypad(makeKeymap(keys), rowPins, colPins, ROWS, COLS);

// define the led pin and button pin
const int ledPin = 13;
const int buttonPin = 10;
// create variables to track the button state
int buttonState = 0;
int lastButtonState = buttonState;

void setup() {
  pinMode(buttonPin, INPUT); // set the button pin as an input
  pinMode(ledPin, OUTPUT); // set the LED control pin as an output
  Serial.begin(9600); // start serial communication
}

void loop() {
 char key = keypad.getKey(); // read the currently pressed key from the keypad
 if(key){ // check if a key is pressed
  // print the value of the pressed key
  Serial.print("KEY,");
  Serial.println(key);
 }
 // read the current state of the buttonPin
 buttonState = digitalRead(buttonPin);
 // check if button was just pressed
 if(buttonState == HIGH && lastButtonState == LOW){
    Serial.println("BTN"); // send the message to the webpage
    delay(200);
    }
  // save the button state for next loop
  lastButtonState = buttonState;
  // if the webpage sent data
  if(Serial.available()){
    String cmd = Serial.readStringUntil('\n'); // read the message from webpage
    cmd.trim(); // remove extra spaces
    if(cmd == "LED"){ //if message says "LED"
      digitalWrite(ledPin, !digitalRead(ledPin)); //turn LED on or off
    }
  }
}
